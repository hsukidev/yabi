## Codebase Patterns
- Project uses Vitest for testing (configured in vite.config.ts with jsdom environment)
- Run tests with `npm run test`, typecheck with `npx tsc -b`, build with `npm run build`
- Test files live in `__tests__` directories next to the source they test
- Utility functions are in `src/utils/` and `src/data/`
