Bulk Select Mode: the Roster's bulk-delete trash icon becomes a Select button, and the action bar gains neutral/accent chrome with an `N SELECTED` pill and a Select all / Clear selection link. Delete now confirms in place (`Delete N? Yes/Cancel`), prunes the deleted mules, and keeps the mode — Cancel is the only exit. On touch, the floating Delete pill morphs into its own inline confirm.
